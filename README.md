easymacconv
Конвертер мак-адрес.
Прога бере мак в довільному форматі та конвертує в інші.
Підтримує формати:
{B0,05,94,F7,D1,61}
B00594F7D161
0xB00594F7D161
B005.94F7.D161
B005-94F7-D161
B0:05:94:F7:D1:61
B0-05-94-F7-D1-61

Історія змін:
macc_0_0_1 - відключено прохання 'Введіть мак' для того, щоб прискорити роботу.
